Alberi di classificazione e previsione

I risultati degli alberi di classificazione bilanciati sono decisamente migliorati. In particolare con i dati sovrcampionati e con sovra/sottocampionamento:

tree.rose <- rpart(y ~ ., data = data.rose)
tree.ov <- rpart(y ~ ., data = data.bal.ov)
tree.un <- rpart(y ~ ., data = data.bal.un)
tree.ou <- rpart(y ~ ., data = data.bal.ou)
tree.smote <- rpart(class ~ ., data = data.sm)

pred.tree.rose <- predict(tree.rose, newdata = test)

pred.tree.ov <- predict(tree.ov, newdata = test)

pred.tree.un <- predict(tree.un, newdata =test)

pred.tree.ou <- predict(tree.ou, newdata = test)

previsto.tree.smote<- predict(tree.smote, newdata = test)
